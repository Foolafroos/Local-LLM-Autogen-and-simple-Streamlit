# Privacy Policy — Local LLM Autogen & Streamlit

**Last updated:** May 2026

## Overview
This project is a local-first Streamlit application for LLM-based multi-agent conversations. All processing happens on your machine.

## Data Handling
- All LLM inference runs locally — no data leaves your machine
- Conversation history is stored in your local SQLite database
- No telemetry, analytics, or tracking

## Your Data
- Conversations and agent outputs live in your local SQLite file
- Delete the database file to clear all data

## Third-Party Services
- None. Everything runs locally.
